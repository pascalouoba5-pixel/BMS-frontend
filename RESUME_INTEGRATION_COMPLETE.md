## 🚀 Comment utiliser

### **1. Démarrer le Backend**
```bash
cd backend
npm start
```

### **2. Démarrer le Frontend**
```bash
cd frontend
npm run dev
```

### **3. Accéder à l'application**
- Frontend : http://localhost:3000
- Backend : http://localhost:5000
